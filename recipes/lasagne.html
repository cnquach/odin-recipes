Lasagne Recipe

Description

This is a very tasty recipe for lasagne, not lasagna.

Ingredients

  • Lasagne sheets
  • Bolognese sauce
  • Bechamel
  • Parmagiano Reggiano

Steps

  1. Make bolognese sauce
  2. Make bechamel sauce
  3. Parboil lasagne sheets
  4. Layer in following order at least 3 times:
    1. Bolognese
    2. Pasta
    3. Bechamel
